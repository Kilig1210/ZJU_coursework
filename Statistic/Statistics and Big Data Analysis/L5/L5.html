连续随机变量#

  • 取值无穷多，不可逐一排列

  • 概率分布 => 概率密度函数 （pdf）,f(x)

\[ \int f(x)dx = 1\]

  • 累积分布函数或分布函数（cdf), F(x)

\[ f(x)= F'(x)\]
\[P(a \leq x \leq b) = F(b)-F(a) = \int_a^{b} f(x)dx \]

  • 对于单点a, \(P(x=a) = F(a) - F(a) =0\)

  • P= \(f(x)\delta\) 为x落在\([x,x+\delta]\)区域的概率, f(x)可以大于1

例：粒子物理中的部分子分布函数（PDF）#

  • 质子中不同类别夸克和胶子（部分子）携带动量比例的概率密度

连续随机变量的期望值和方差#

\[ E(x) = \int x f(x) dx \]
\[ E(g(x)) = \int g(x) f(x) dx \]
\[ \text{var}(x) = \int (x-E(x))^2 f(x) dx = E(x^2)-(E(x))^2\]

连续均匀分布(Uniform)#

(1)#\[\begin{equation} f(x) = \begin{cases} \frac{1}{b-a} & a\leq x \leq b \\ 0 & \text{其他} \end{cases} \end{equation}\]
\[ E(x)= \int_a^{b} x \frac{1}{b-a} dx = \frac{b^2-a^2}{2(b-a)} = \frac{a+b}{2}\]
\[ \text{var}(x) = \int_a^{b} \frac{x^2}{b-a}dx - (\frac{a+b}{2})^2 = \frac{b^3-a^3}{3}-(\frac{a+b}{2})^2 = \frac{(b-a)^2}{12}\]

指数分布 (Exponential)#

(2)#\[\begin{equation} f(x)= \begin{cases} \lambda e^{-\lambda x} & x>0 \\ 0 & x\leq0 \\ \end{cases} \end{equation}\]

  • 描述某事件发生所需要的时间

  • 单位时间事件发生的次数平均值为\(\lambda\),时长x之内发生的总次数\(\lambda\)x. 实际观察到的次数符合泊松分布

\[ g(n) = (\lambda x)^n\frac{e^{-\lambda x}}{(n)!}\]

  • 第一次观察到事件发生需要等待时间x，计算其概率

    • 在时间\(0-x\)之间，发生事件次数0，对应概率

    \[ g(0) = (\lambda x)^0 \frac{e^{-\lambda x}}{(0)!}= e^{-\lambda x}\]

    • 在时间\(x-x+dx\)之间，发生事件1次，对应概率

    \[ g(1) = (\lambda dx)^1 \frac{e^{-\lambda dx}}{(1)!} = \lambda dx \]

    • 总概率

    \[ f(x)dx = g(0)g(1) = \lambda e^{-\lambda x} dx \]

  • 期望值: \(1/\lambda\), 方差 \(1/{\lambda}^2\)

  • 更详细的内容参见《概率导论》第六章,伯努利过程和泊松过程

正态分布(Normal)/高斯分布(Gaussian)#

\[ f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}\]

  • 记为 \(X\sim N(\mu,\sigma)\), \( N(0,1)\)标准正态分布

  • 标准正态分布的累积分布函数（cdf）,标记为\(\Phi(x)\)

\[ \Phi(-x) = 1- \Phi(x)\]

二维正态分布#

\[ f(x,y) = \frac{1}{\sqrt{1-\rho^2}2\pi\sigma_1\sigma_2}exp (-\frac{1}{2(1-\rho^2)}(\frac{(x-\mu_1)^2}{\sigma_1^2}+\frac{(y-\mu_2)^2}{\sigma_2^2}-\frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2}))\]
\[\begin{split}\Sigma= \begin{bmatrix} \sigma_1^2 & \rho \sigma_1\sigma_2 \\ \rho \sigma_1\sigma_2 & \sigma_2^2 \end{bmatrix}\end{split}\]
\[ f(x,y) = \frac{1}{\sqrt{(2\pi)^k \det \Sigma}} \exp{\bigl(-1/2 (x-\mu)^\intercal \Sigma ^{-1} (x-\mu)\bigl)}\]

import numpy as np
import matplotlib.pyplot as plt

协方差和相关系数#

  • Cov(X,Y)=\(E[(x-\mu_1)(y-\mu_2)] = E(xy)-\mu_1\mu_2\)

  • 若X,Y独立，则Cov(X,Y)=0

  • 相关系数 Corr(X,Y) = Cov (X,Y)/\(\sigma_1 \sigma_2\)

  • 二维正态分布的Cov(X,Y) = \(\rho \sigma_1 \sigma_2\), Corr(X,Y) = \(\rho\)

  • 记\(X_1,\cdots,\)为均值 E[X] ,方差为 var(X) 的同分布随机变量

  • 记N为取值非负的随机变量,有均值 E[N] ,方差 var(N)，若

\[Y= X_1 + X_2 + \cdots X_N\]

那么

\[ E(Y) = E(N)E(X)\]
\[ \text {var}(Y) = \text {var}(X)E(N)+ (E(X))^2\text {var}(N) \]

  • 他在每家书店停留的时间服从指数分布，\(\lambda=0.5\)。

  • 他会一直逛书店，直到买到书为止。

记他为了买书所花费的时间总和为Y，请用公式计算Y的

  • 均值

  • 方差
