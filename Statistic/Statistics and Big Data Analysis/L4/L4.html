随机变量#

  • 样本空间：所有试验结果的集合 实验结果 => 实数

  • 随机变量：试验结果的实数函数

例：连续抛硬币5次的结果 - 长度为5的正反序列：不是随机变量 - 5次抛掷正面朝上的次数：是随机变量

  • 离散随机变量：随机变量的取值范围是有限集合

  • 离散随机变量的函数也是离散随机变量

分布列#

  • 分布列：对于随机变量的每个取值，给出一个概率

\[ p_X(x) = P(\{X=x\})\]

  • X随机变量，x随机变量的取值，\(p_X\) 随机变量X的分布列

\[ \sum_x p_X(x) = 1 \]

  • 例：一枚硬币抛两次，X为正面朝上的次数

(1)#\[\begin{equation} p_X(x) = \begin{cases} 1/4 & \text{x=0 或 x=2}\\ 1/2 & \text{x=1}\\ 0 & \text{其他} \end{cases} \end{equation}\]

分布函数#

  • 分布函数 (累积分布函数 CDF)：对随机变量X，取值小于x的概率

\[ F(x)=P(X<=x) = \sum_{i|a_i<=x} p_i\]

  • 对任意随机变量X, F(x)单调递增, \(x\rightarrow \infty, F(x) \rightarrow 1\)

(2)#\[\begin{equation} p_X(x) = \begin{cases} p & \text{k=1}\\ 1-p & \text{k=0} \end{cases} \end{equation}\]

二项随机变量(Binomial)#

  • 一个伯努利变量在n次试验中得到特定结果的次数k

\[p(k) = \binom{n}{k} p^{k} (1-p)^{n-k}\]

其中

\[\binom{n}{k}= \frac{n!}{k!(n - k)!} = C_{n}^k\]

  • 二项分布的参数：n,p, 随机变量k。用 B(n,p)表示

期望值和方差#

随机变量的数字特征

  • 期望值：X所有取值的概率加权平均 (均值)

\[ E(X) = \sum_x x p_X{x}\]

  • 方差：X在期望值周围分散程度

(3)#\[\begin{align} \text{var}(X) &= E[(X-E(X))^2] \\ &=E(X^2 + E(X)^2 -2XE(X)) \\ &=E(X^2) - (E(X))^2 \end{align}\]

  • 标准差：方差的平方根

\[ \sigma_X = \sqrt{\text{var}(X)}\]

  • n阶矩

\[ E[X^n] = \sum_x x^n p_X(x) \]
- 一阶矩：期望值
- 二阶矩：可算出方差

期望值和方差的性质#

  • 若Y为X的线性函数，Y=aX+b

\[ E(Y) = aE(X)+b \]
\[ \text{var}(Y) = a^2\text{var}(X)\]

  • 若g(X)不是X的线性函数，\(E(g(X))\neq g(E(X))\)

  • 例: 平均速度和平均时间。假设上学路途共2公里，天气好时（概率0.6），步行上学，速度5公里/小时，天气不好时，骑车上学，速度30公里/小时。求上学平均用时。

    • t = 2/v = g(v)

    • 求E(t)

    \[ \sum_t tp(t) = 2/5 *0.6 + 2/30*0.4 = 4/15\]

    • 若用 2/E(v) 来计算

    \[\begin{split} E(v) = 5*0.6 + 30*0.4 = 15,\\ 2/E(v) = 2/15 \end{split}\]

常见随机变量的期望值和方差#

均匀分布#

  • 变量X可在[a,b]范围内取任意整数

(4)#\[\begin{equation} p_X(k) = \begin{cases} 1/(b-a+1) & k=a,a+1,\cdots,b \\ 0 & \text{其他} \end{cases} \end{equation}\]
\[ E(X) = 1/(b-a+1) \sum_{k=a}^{b} k = (a+b)/2\]
\[ E(X^2) = 1/(b-a+1) \sum_{k=a}^{b} k^2 \]

  • 若 a=1, b=n

\[ E(X^2) = \frac{1}{n} \sum_{k=1}^{n} k^2 = \frac{1}{6}(n+1)(2n+1)\]
\[ \text{var}(X) = E(X^2)-(E(X))^2 = \frac{n^2-1}{12} \]

  • 平移 a=1, b=n 至 [a,b]区间，保持区间长度相等，方差相同，n=b-a+1

\[ \text{var}(X) = \frac{(b-a)(b-a+2)}{12} \]

伯努利分布#

(5)#\[\begin{equation} p_X(x) = \begin{cases} p & \text{k=1}\\ 1-p & \text{k=0} \end{cases} \end{equation}\]
\[ E(X) = p\cdot 1 + (1-p)\cdot 0 = p\]
\[ E(X^2) = p\cdot 1 + (1-p)\cdot 0 = p\]
\[ \text{var}(X) = E(X^2) - (E(X))^2= p-p^2= p(1-p)\]

利用期望值进行决策#

  • 一个问题的多个方案，选择具有最大期望回报的策略

  • 例：智力游戏：2个问题，问题1简单，回答正确的概率是0.8，可以得到100元；问题2难，回答正确的概率是0.5，可以得到200元。

需要选定答题顺序，但第一题答对了才能答第二题。问应该先答哪一题？

  • 策略1：先答第一题, E(X) = 0.8 * 100+ 0.8 * 0.5 * 200 = 160

  • 策略2：先答第二题：E(X) = 0.5 * 200 + 0.5 * 0.8 * 100 = 140

条件期望值和方差#

  • 条件概率： P(X|Y），在给定条件Y下，也可以计算X的期望值和方差

\[ E(X|Y=y) = \sum_x x p(x|Y=y)\]

  • 全概率公式：

    (8)#\[\begin{align} E(X) &= \sum_x x p(x)\\ &=\sum_x \sum_y xp(x|y)p(y)\\ &=\sum_y E(X|Y=y)p_Y(y) \end{align}\]

几何分布的期望值和方差#

  • 写一个程序成功的概率是p，连写X次直到成功。X的期望值和方差？

    • X符合几何分布

    \[ p(X) = (1-p)^{x-1}p\]
    \[ E(X) = \sum_x x(1-p)^{x-1}p \]

    • 全概率公式：

      • 条件A：写一次就成功，X=1

      • 条件\(A^c\): 写大于1次成功, X>1

    \[ E(X|x=1) = 1\]
    \[ E(X|x>1) = ? \]

    • 坐标变换: x-1 => y, x>1 => y>0

      E(X|x>1) => E(Y+1|y>0) = E(Y+1) = E(Y)+1, y>0

    • 总期望值:

      (9)#\[\begin{align} E(X) &= p(x=1)E(X|x=1)+ p(x>1)E(X|x>1)\\ &= p\cdot 1 + (1-p)(E(X)+1)\\ &= 1/p \end{align}\]

    • 方差：

      (10)#\[\begin{align} E(X^2) &= p(x=1)E(X^2|x=1)+ p(x>1)E(X^2|x>1)\\ &= p\cdot 1 + (1-p)(E(X+1)^2)\\ &= p+(1-p)(E(X^2)+2E(X)+1) \\ &= \frac{2}{p^2}-\frac{1}{p}\\ \text{var}(X) &= E(X^2)-E(X)^2 = \frac{1-p}{p^2} \end{align}\]
