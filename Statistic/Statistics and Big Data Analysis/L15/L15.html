对于上面的例子

  • 特征值（输入）有两个\(\{x_1,x_2\}\)

  • 输出为y，取值为0或1

  • 假设函数 y = h(x)未知

  • 代价函数 L 未知

边界#

  • 划分类别的线性边界为

\[ x_2 = c_0 + c_1 x_1\]

  • 也可写为

\[ - c_0 - c_1 x_1 + x_2= 0 = z\]

  • 二元分类

    • \(z > 0\)，类别y=1

    • \(z <= 0\)，类别y=0

假设函数#

  • 与单特征值类似，构建logistic函数

    \[ f(z) = \frac{1}{1+e^{-z}}\]

    • \(f(z)\)为给定测量值z时，类别\(y=1\)的概率

    • \(1-f(z)\)为类别为\(y=0\)的概率

    • \(P(y=1|z)=f(z)\)

    • \(P(y=0|z)= 1- f(z)\)

    \[\begin{split} y=h(x)=\begin{cases} 1,& P(y=1)>P(y=0) \\ 0, & P(y=1)\leq P(y=0) \end{cases} \end{split}\]

    • \(f(z)>1-f(z)\) => \(f(z)>0.5\)

    \[\begin{split} y=h(x)=\begin{cases} 1,& f(z)>0.5 \\ 0, &f(z)\leq 0.5 \end{cases} \end{split}\]

代价函数#

\[ \sum_{i=1}^{N} y_i\log\frac{1}{1+e^{-z_i}} + (1-y_i)\log\frac{e^{-z_i}}{1+e^{-z_i}}\]

  • 找出使得代价函数最小的参数\(c_0,c_1\)的值（阶梯下降法）

  • 写为更一般的表达形式，对于多个特征值\(\{x_1,\cdots,x_n\}\)，线性边界为

\[ z = c_0+ c_1 x_1 + \cdots + c_n x_n\]

  • 二元分类： $\( y=h(x)=\begin{cases} 1,& f(z)>0.5 \\ 0, &f(z)\leq 0.5 \end{cases} \)$

  • 找到使得损失函数最小的参数\(\{c_0,\cdots,c_n\}\)

要素#

  • 输入：{\(x_1,x_2\)}

  • 输出：y=0 or 1 or 2

  • 假设函数: y=h(x)

如何确定假设函数？回顾二元分类

  • 对每个测量值(x_{1},x_{2}) 对测量值建立超空间(hyperspace)

\[ z=f(x_{1},x_{2}) \]

  • 建立测量值z下属于类别y=1和y=0的概率

\[P(y=1 | z)=\text{sigmoid}(z)\]
\[P(y=0|z)=1-\text{sigmoid}(z)\]

  • 类别鉴定

\[\begin{split} y=\begin{cases} 1,& P(y=1)>P(y=0) \\ 0, &P(y=1)\leq P(y=0) \end{cases} \end{split}\]

推广至多元

  • 概率

\[\begin{split} \begin{align} &P(y=2 | z_2)=\text{sigmoid}(z_2): y=2与y=0的比较\\ &P(y=1 | z_1)=\text{sigmoid}(z_1): y=1与y=0的比较\\ &P(y=0|z_?) =? \end{align} \end{split}\]

  • 分类

\[\begin{split} y=\begin{cases} 2,& max(P(y=k))=P(y=2) \\ 1,& max(P(y=k))=P(y=1) \\ 0,& max(P(y=k))=P(y=0) \end{cases} \end{split}\]

可以表示为 \(y=\text{argmax}(P_k)\)

概率函数#

\[\begin{split} \begin{align} &P(y=2 | z_2)=\text{sigmoid}(z_2): y=2与y=0的比较\\ &P(y=1 | z_1)=\text{sigmoid}(z_1): y=1与y=0的比较\\ &P(y=0|z_?) =? \end{align} \end{split}\]

  • 列出已知限定条件

\[\sum_{k=0}^{m} P(y=k) =1\]

  • 在二元分类下(k vs 0)

\[ P(y=k)= \frac{1}{1-e^{-z_k}}\]
\[ P(y=0)= \frac{e^{-z_k}}{1-e^{-z_k}}\]
\[ \frac{P(y=k)}{P(y=0)} = e^{z_k}\]

  • 加和所有类别项

\[\begin{split} \begin{align} \sum_{k=1}^{m} \frac{P(y=k)}{P(y=0)} &=\sum_{k=1}^{m} e^{z_k} \\ &= \frac{1-P(y=0)}{P(y=0)}\\ &=\frac{1}{P(y=0)}-1 \end{align}\end{split}\]
\[P(y=0)=\frac{1}{1+\sum_{k=1}^{m} e^{z_k}}\]

  • 若认为\(z_0=0\), 则可写为

\[P(y=0)=\frac{1}{\sum_{k=0}^{m} e^{z_k}}\]

  • 对任意的k，有

\[P(y=k)=\frac{e^{z_k}}{\sum_{k=0}^{m} e^{z_k}}\]

  • 对比二元分类

\[P(y=k)=\frac{1}{1+ e^{-z_k}}=\frac{e^{z_k}}{1+ e^{z_k}}\]

在分母上多了其他类别的贡献

softmax函数的性质#

  • 概率函数 \(P(y=k)=\frac{e^{z_k}}{\sum_{k=0}^{m} e^{z_k}}\) 也被称为softmax函数

  • 它会放大一组\(z_k\)数值中的最大值

from scipy.special import softmax
x=np.linspace(-10,10,40)
plt.plot(x,softmax(x),marker="x")

[<matplotlib.lines.Line2D at 0x16edd4dc0>]

  • 函数对于数组的平移不敏感

    \[P(y=k)=\frac{e^{z_k}}{\sum_{k=0}^{m} e^{z_k}}\]

    • \(z_k => z_k + c\), 函数值不变

  • 对于求解系列的\(z_k\)，解不唯一

代价函数#

  • 对每个事例

    • \(z_k = c_{0k}+c_{1k}x_1+c_{2k}x_2\)

    • \(P_k = \text{softmax}(z_k)\)

    • \(y' = \text{argmax}(P_k)\), \(y'\)表示估计值，y表示真实值

  • 对所有事例求似然值

\[ L = \Pi_{i=1}^{n} \sum_{k=0}^{m} P_{ik} \cdot (y_i==k) \]
\[ -\ln L = -\sum_{i=1}^{n}\sum_{k=0}^{m} \ln P_{ik} \cdot (y_i==k)\]

  • 如何数学表示\(y_i==k\)?

    • OneHot矩阵表示方法

    • 用一维矩阵表示单个事例

      • y=2: [0, 0, 1]

      • y=1: [0, 1, 0]

      • y=0: [1, 0, 0]

    • 对于n个事例: Y=nx3矩阵，每行代表一个事例, \(y_i==k\) => \(y_{ik}=1\)

  • 矩阵表示P：P=nx3矩阵

  • \(\sum_{i=1}^{n}\sum_{k=0}^{m} \ln P_{ik} \cdot (y_i==k)\) 矩阵表示：

\[\sum_{i=1}^{n}\sum_{k=0}^{m}\ln P_{ik}Y_{ik} = \text{Tr}(\ln P\cdot Y^T)\]

P矩阵#

  • 对每个事例

    • \(z_k = c_{0k}+c_{1k}x_1+c_{2k}x_2\)

    • \(P_k = \text{softmax}(z_k)\)

  • Z矩阵: n x 3

  • P矩阵: n x 3

  • X矩阵：n x 2，2为特征值个数

  • C矩阵：2 x 3，2为特征值个数，3为类别

  • Z = \(X\cdot C\) + \(C_0\), \(C_0\)为 1x3 矩阵

  • \( L = -\text{Tr}(\ln P\cdot Y^T) = -\text{Tr}(\ln [\text{softmax}(X\cdot C + C_0)]\cdot Y^T)\)

阶梯下降法#

  • 代价函数求导

\[\frac{\partial L} {\partial C} = X^T \cdot (Y - P) \]
\[\frac{\partial L} {\partial C_0} = 1^T \cdot (Y - P) \]

  • \(C' = C- \alpha \frac{\partial L} {\partial C}\)

  • \(C_0' = C_0- \alpha \frac{\partial L} {\partial C_0}\)
