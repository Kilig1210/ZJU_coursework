数据集#

  • 一般形式：二维数组，(n_samples x n_features)，数据(Samples)，特征（features）

  • 监督学习 (Supervised learning)：数据集中既含有特征，也含有标签（label）。学习标签下的特征，给定新数据的特征，判断标签

  • 标签：离散或连续，分类或回归

  • 数据集拆分：为了评估效果，通常拆分为训练（train）和验证（test）两部分

from sklearn.inspection import DecisionBoundaryDisplay
from sklearn import svm, metrics
from sklearn.model_selection import train_test_split
from sklearn.datasets import load_iris
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt


iris = load_iris()

iris_df = pd.DataFrame(np.c_[iris.data,iris.target], columns=iris.feature_names+["target"])

X_train, X_test, y_train, y_test = train_test_split(
    iris_df[["petal width (cm)","petal length (cm)"]], iris.target, test_size=0.5, shuffle=True
)

X=X_train
y=y_train
print(X)
C = 1.0  # SVM regularization parameter
models = (
    svm.SVC(kernel="linear", C=C),
    svm.LinearSVC(C=C, max_iter=10000),
    svm.SVC(kernel="rbf", gamma=0.7, C=C),
    svm.SVC(kernel="poly", degree=3, gamma="auto", C=C),
)
models = (clf.fit(X, y) for clf in models)
titles = (
    "SVC with linear kernel",
    "LinearSVC (linear kernel)",
    "SVC with RBF kernel",
    "SVC with polynomial (degree 3) kernel",
)
fig, sub = plt.subplots(2, 2,figsize=(10,10))
plt.subplots_adjust(wspace=0.4, hspace=0.4)

X0, X1 = X_train["petal width (cm)"], X_train["petal length (cm)"]
X2, X3 = X_test["petal width (cm)"], X_test["petal length (cm)"]

for clf, title, ax in zip(models, titles, sub.flatten()):
    disp = DecisionBoundaryDisplay.from_estimator(
        clf,
        X,
        response_method="predict",
        cmap=plt.cm.Set3,
        alpha=0.8,
        ax=ax,
        xlabel="petal width (cm)",
        ylabel="petal length (cm)",
    )
    ax.scatter(X0, X1, c=y, cmap=plt.cm.Set3, s=40, edgecolors="k")
    ax.scatter(X2, X3, c=y_test, cmap=plt.cm.Set3, s=20, marker="D",edgecolors="grey")

    ax.set_xticks(())
    ax.set_yticks(())
    ax.set_title(title)

学习的性能量化#

  • Accuracy: 所有样本中判断正确的比例

Iris数据中,70/75=0.93

  • 为什么有时Accuracy不足以判断机器学习的性能？

例：两类判断，没有学习，认为所有的数据类别均为A。实际数据中A类有95个，B类有5个，Accuracy=95/100=0.95

  • Precision: 被判为A类的样本中正确的比例

上例中，对类别A, precision=0.95, 对类别B，不存在

  • Recall (Sensitivity): 真实为A类的样本中正确被鉴别的比例

上例中，对类别A，Recall=1, 类别B Recall=0

import numpy as np
import scipy.stats as st
import matplotlib.pyplot as plt

fig, ax = plt.subplots(1, 2,figsize=(10,3))

x=np.linspace(-8,8,200)
pdfa=st.norm.pdf(x,-3,1.5)
pdfb=st.norm.pdf(x,3,1.5)

ax[0].plot(x,pdfa,x,pdfb)
ax[0].fill_between(x,pdfa,where=x>=0,alpha=0.3)
ax[0].fill_between(x,pdfb,where=x>=0,alpha=0.3)

ax[1].plot(x,pdfa,x,pdfb)
ax[1].fill_between(x,pdfb,where=x<=0,alpha=0.3)
ax[1].fill_between(x,pdfb,where=x>=0,alpha=0.3)
ax[0].set_title("Precision")
ax[1].set_title("Recall")

Text(0.5, 1.0, 'Recall')

  • F1-score: 很多时候precision和recall无法兼得，accuracy不能完全衡量性能，赋予precision和recall相同的权重

    F1_score = 2 precision * recall/(precision+recall)
