回归#

  • 高尔顿(1886年)研究父母与子女的身高关系，父母身高Y，子女身高X

  • Y与X可以用线性函数来描述

  • 若考虑\(Y-\bar{Y}\)与\(X-\bar{X}\)的关系，对于极端的\(Y-\bar{Y}\)值，\(|X-\bar{X}|\)会变小

  • 均值回归 （Regression to the mean）

    \[ X= aY +b\]
    \[ \bar{X}= a\bar{Y} +b\]
    \[ X- \bar{X} = a (Y-\bar{Y})\]

  • 对于变量Y，想找出影响它的所有\(X_i\)，并用函数描述它

\[ Y = f(X_1,\cdots,X_p)+e\]

  • e为未考虑的其他随机因素，要求它均值为0，E(e)=0

  • \(f(X_i)\)称为Y对\(X_i\)的回归函数

\[ E[Y-f(X_i)]^2 = E(e^2) = \text{Var}(e) = \sigma^2\]

  • \(\sigma^2\)越小，用f逼近Y导致的均方误差越小

回归分析的应用

  1. 描述观测到的一组X和Y的关系

  2. 估计回归函数

  3. 预测Y值

通常假设，X的测量没有误差，而Y受到e的因素影响有随机误差

一元线性回归#

  • 对于一组n个测量(\(X_i,Y_i\))值，认为Y与X符合线性关系

\[ Y_i = b_0+b_1X_i+e_i \]

  • \(b_0\)截距，\(b_1\)斜率

  • Y的方差为: var(Y) = var(e) = \(\sigma^2\)

  • 如何估计\(b_0\)和\(b_1\)?

最小二乘法（Least Squares）#

希望Y的预测值与测量值偏差越小越好,对于n个测量

\[ Q = \sum (Y_i - (b_0+b_1X_i))^2\]

最小化Q,最小二乘法

将回归函数的表达式改写为

\[ Y = \beta_0 + \beta_1 (X-\bar{X})\]

上式被称为模型的中心化

\[ Q = \sum [Y_i - (\beta_0+\beta_1(X_i-\bar{X}))]^2 \]

对\(\beta_0\)和\(\beta_1\)求偏导

\[ \frac{\partial Q}{\partial \beta_0} = -2 \sum [Y_i-\beta_0-\beta_1(X_i-\bar{X})] =0\]
\[ \frac{\partial Q}{\partial \beta_1} = -2 \sum [Y_i-\beta_0-\beta_1(X_i-\bar{X})](X_i-\bar{X}) =0\]

得到

\[ \hat{\beta_0} = \sum Y_i/n = \bar{Y}\]
\[ \hat{\beta_1} = \frac{\sum (X_i -\bar{X})Y_i}{S^2_X}\]

\(\hat{\beta_0}, \hat{\beta_1}\)均为\(Y_i\)的线性函数

  • \(\hat{\beta_0}, \hat{\beta_1}\)是\(\beta_0,\beta_1\)的无偏估计

\[ \text{var}(\hat\beta_0) = \text{var}(\bar{Y}) = \sigma^2/n \]
(1)#\[\begin{align} \text{var}(\hat\beta_1) &= \text{var}(\frac{\sum (X_i -\bar{X})Y_i}{S^2_X})\\ &= \sum(\frac{ (X_i -\bar{X})}{S^2_X})^2\text{var}(Y_i)\\ &= \text{var}(Y_i)/S^2_X = \sigma^2/S^2_X \end{align}\]

  • 截距\(\beta_0\)的方差由测量数据的样本决定

  • \(\beta_1\)的方差由\(S^2_x\)决定，X的点取得越紧密，方差越大，应尽量取分散的点

  • 可以证明，\(\beta_0\)和\(\beta_1\)的协方差cov\((\beta_0,\beta_1)=0\)

(2)#\[\begin{align} Y &= \beta_0 + \beta_1 (X-\bar{X}) \\ b_0 &= \beta_0 - \beta_1\bar{X}\\ b_1&= \beta_1\\ \text{var}(b_0) &= \text{var}(\beta_0) + \bar{X}^2\text{var}(\beta_1)\\ \text{var}(b_1) &= \text{var}(\beta_1) \end{align}\]

残差与\(\sigma^2\)的估计#

参数估计的Y与测量值Y的差，

\[ \delta_i = Y_i - \hat{Y}_i\]

称为残差。是最小二乘法中的分量

  • 残差用于检验回归函数与数据的符合程度，不随x有特定趋势

  • 残差用于估计\(\sigma^2\)

\[ Y = b_0+b_1X + e\]
\[ \text{var}(Y) = \text{var}(e) = \sigma^2\]

  • 如何估计\(\sigma^2\)?

\[ \hat{\sigma}^2=\sum \delta_i ^2 / (n-?)\]

  • 分母为n-2, 证明请参照概率论与数理统计6.2.2章。2个未知参量

  • 当\(e_i\)符合正态分布N(\(0,\sigma^2\))时，\(\hat{\sigma}^2/\sigma^2 \sim \chi^2_{n-2}\)

    • 可以用于估计拟合与实验数据的符合程度

import numpy as np
import seaborn as sns

b=4

y=[np.random.normal(a*i+b,5) for i in x]

from scipy.stats import linregress

#sx=sum (x-xi)^2
sx=np.var(x)*len(x)
sy=np.var(y)*len(y)
meanx=np.mean(x)
meany=np.mean(y)
#b1=beta1
b1= sum((x-meanx)*y)/sx
#b0=beta0-b1*xbar=ybar-b1*xbar
b0= meany-b1*meanx
residual=sum((y-b0-b1*x)**2)
varb1 = residual/sx/(len(x)-2)
varb0 = residual/len(x)/(len(x)-2)+meanx**2*varb1
print("Calculation: slope=",b1,"err=",np.sqrt(varb1))
print("Calculation: intercept=",b0,"err=",np.sqrt(varb0))
result=linregress(x,y)
print("Fit: slope=", result.slope,result.stderr)
print("Fit: intercept=",result.intercept,result.intercept_stderr)
